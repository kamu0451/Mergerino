Windows desktop chat client

Merged live chat for the channels you watch.

Mergerino combines a Chatterino-style multi-channel workflow with support for Twitch, Kick, YouTube, and TikTok LIVE in one practical desktop app.

Built for watching and moderating live chat.

Mergerino focuses on the chat workflows users expect in a desktop client: multiple channels, readable messages, platform account support, and direct access to live chat actions where each platform allows them.

Multi-platform chat

Open Twitch, Kick, YouTube, and TikTok LIVE channels in one desktop app.

YouTube OAuth

Google sign-in is used only for visible YouTube chat features such as sending messages and permitted moderation actions.

Local desktop app

Mergerino does not run a hosted backend for YouTube login. OAuth tokens are stored locally on the user's device.

Google user data use

Mergerino uses Google user data only to provide the YouTube chat features shown in the app. It does not sell Google user data, use it for advertising, or share OAuth tokens with the maintainer.

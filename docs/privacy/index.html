Privacy Policy

Mergerino is a Windows desktop chat client for viewing and participating in live chat across supported streaming platforms. Mergerino does not operate a hosted backend for Google or YouTube login. When you sign in with Google, the app opens Google's OAuth flow and uses the YouTube Data API directly from your desktop device so YouTube chat features can work inside the app.

Google user data accessed

Mergerino requests the https://www.googleapis.com/auth/youtube.force-ssl OAuth scope because YouTube requires an authenticated YouTube Data API scope for the live chat write and moderation actions Mergerino provides. This scope may allow broader YouTube account permissions, but Mergerino uses it only for the YouTube chat features described in this policy.

Depending on which YouTube features you use, Mergerino may access or process the following Google user data:

  • OAuth authorization data, including access tokens, refresh tokens, token expiration time, and the OAuth client ID used for sign-in
  • Your authenticated YouTube channel ID and channel display name, used to show which YouTube account is connected
  • Live stream and live chat identifiers needed to connect to the chat you open, including YouTube video IDs, live chat IDs, stream title, stream start time, viewer count, thumbnail or preview URL, and owner channel ID
  • Live chat content displayed in the app, including message IDs, author display names, author channel IDs, message text, timestamps, badges, emoji, membership events, Super Chat or sticker details, and other chat metadata returned by YouTube for the live chat you opened
  • Chat messages you choose to send through Mergerino as the signed-in YouTube account
  • Moderation data for actions you choose to perform, including the target message ID, target user's channel ID and display name, timeout duration, ban status, and ban IDs returned by YouTube

How Google user data is used

Mergerino uses Google user data only to provide and improve the visible YouTube chat features in the desktop app. Specifically, Mergerino uses Google user data to:

  • Connect your YouTube account and show the connected account in the app
  • Refresh OAuth access so you do not have to sign in every time you open Mergerino
  • Find the active live chat for a YouTube stream you open
  • Display YouTube live chat messages, author information, chat badges, emoji, membership events, Super Chats, stickers, stream status, and stream metadata in the app
  • Send chat messages that you type and submit through Mergerino
  • Perform moderation actions that you explicitly choose, such as deleting a live chat message, timing out a user, banning a user, or unbanning a user when YouTube permits the signed-in account to do so
  • Show YouTube error or status messages related to sign-in, chat, sending, and moderation

Mergerino does not use Google user data for advertising, tracking, credit decisions, lending, data brokerage, selling user data, unrelated analytics, or training generalized AI or machine learning models. Mergerino does not intentionally access, upload, edit, or delete YouTube videos, ratings, comments outside live chat, captions, playlists, subscriptions, or analytics.

Storage and retention

Mergerino stores the connected YouTube account's display name, channel ID, OAuth client ID, access token, refresh token, and token expiration time locally in the app's settings on your device. This local storage is used to keep you signed in between app launches.

Live chat messages and stream metadata are displayed in the app while you use YouTube chat features. Mergerino does not upload those messages or metadata to a Mergerino server. Ban IDs returned by YouTube may be cached in memory during the current app session so Mergerino can unban users it banned during that same session.

If you remove a YouTube account from Mergerino, Mergerino removes the locally saved YouTube account fields for that account. You can also revoke Mergerino's Google access at any time from your Google Account permissions page: https://myaccount.google.com/permissions. Revoking access invalidates Mergerino's OAuth tokens for your Google account.

Sharing and disclosure

Google user data is sent directly between your desktop device and Google or YouTube services to provide the YouTube features you request. Mergerino does not operate a server that receives your Google OAuth tokens or Google user data. The Mergerino developer does not receive, store, sell, rent, or share your Google OAuth tokens or Google user data.

Mergerino does not transfer Google user data to advertising platforms, data brokers, information resellers, analytics providers, or other third parties. Mergerino does not allow humans to read your Google user data through any Mergerino backend because no such backend is operated for YouTube login or chat.

Protection of Google user data

Mergerino sends Google OAuth and YouTube Data API requests over HTTPS. OAuth token request bodies are hidden from Mergerino's network request logging. Google OAuth tokens are stored locally on your device and are not transmitted to the Mergerino developer. You are responsible for protecting your Windows account, local device, backups, and any copied Mergerino settings files.

Revoking access

You can revoke Mergerino's Google access at any time from your Google Account permissions page: https://myaccount.google.com/permissions.
